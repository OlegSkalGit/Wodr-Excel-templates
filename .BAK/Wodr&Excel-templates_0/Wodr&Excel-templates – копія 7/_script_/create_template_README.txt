ІНСТРУКЦІЯ ДО СКРИПТА: create_template.py (СПРОЩЕНА ВЕРСІЯ)
==========================================================

Цей скрипт автоматично створює все необхідне для автоматизації документів, порівнюючи дві версії файлу.

1. ПРИЗНАЧЕННЯ
--------------
Вам потрібно лише вказати "Старий" та "Новий" файли. Скрипт сам створить у тій же папці:
1. Шаблон (.docx або .xlsx) зі змінними {{ field_1 }}, {{ field_2 }}...
2. Конфігураційний Excel (Config) з уже заповненими даними з "Нового" файлу.
3. BAT-файл для швидкого запуску генерації в один клік.

2. ЯК ЗАПУСТИТИ
---------------
Відкрийте термінал у папці _script_ та виконайте:

python create_template.py "<шлях_до_старого_файлу>" "<шлях_до_нового_файлу>"

Приклад:
python create_template.py "C:\Docs\Dogovor_V1.docx" "C:\Docs\Dogovor_V2.docx"

3. РЕЗУЛЬТАТИ (Створюються в папці старого файлу)
------------------------------------------------
Якщо ваш файл називався "Dogovor_V1.docx", скрипт створить:
- Dogovor_V1_template.docx  (Файл-шаблон зі змінними)
- Dogovor_V1_config.xlsx    (Ексель з даними та налаштуваннями)
- Dogovor_V1_run_all.bat    (Файл для запуску генерації)

4. ПОРЯДОК РОБОТИ
-----------------
1. Запустіть create_template.py з двома файлами.
2. Перейдіть у папку, де лежить ваш оригінальний файл.
3. Ви знайдете там файл "..._config.xlsx". Відкрийте його і додайте нові рядки з даними, якщо потрібно згенерувати більше документів.
4. Двічі клікніть на "..._run_all.bat". Скрипт автоматично запустить main.py і створить документи на основі ваших даних.

5. ВАЖЛИВІ ЗАУВАЖЕННЯ
---------------------
- Шляхи: Скрипт приймає як абсолютні, так і відносні шляхи.
- Відносні шляхи рахуються від поточної папки, де ви відкрили термінал.
- Автоматизація: BAT-файл автоматично знаходить потрібну версію Python у папці _script_\.venv.
