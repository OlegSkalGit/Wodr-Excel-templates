========================================================================
ІНСТРУКЦІЯ: create_template.py — Автоматичне створення шаблонів
========================================================================

Цей скрипт призначений для інтелектуального порівняння двох файлів 
(Word або Excel) та автоматичного створення системи генерації документів.

ЯК ЦЕ ПРАЦЮЄ:
1. Ви надаєте два файли: "Старий" (оригінал) та "Новий" (зразок зі змінами).
2. Скрипт знаходить відмінності та створює три файли у папці оригіналу:
   - _template: Шаблон із вставленими змінними {{ field_N }}.
   - _config: Excel-файл для керування даними.
   - _run_all.bat: Файл для миттєвого запуску генерації.

ПАКЕТНА ОБРОБКА (СКАНУВАННЯ КАТАЛОГУ):
Якщо ви вкажете третій параметр — шлях до папки, скрипт просканує її та:
- Знайде всі файли такого ж типу.
- Автоматично витягне з них дані за знайденими координатами змінних.
- Додасть ці дані як нові рядки в Excel-конфіг.
* Вихідні файли (Старий та Новий) автоматично ігноруються при скануванні.

ГОЛОВНІ ОСОБЛИВОСТІ:
- Smoothing (Згладжування): Об'єднує імена та довгі рядки в одну змінну.
- Токенізація: Слова та числа не розбиваються по літерах.
- Excel-безпека: Формули та типи даних (числа, дати) зберігаються.
- Форматування Word: Стилі не втрачаються завдяки групуванню runs.

ВИКОРИСТАННЯ:
python create_template.py "Старий" "Новий" [каталог_сканування]

СТРУКТУРА EXCEL-КОНФІГУ:
- Рядок 5 (OLD): Дані зі "Старого" файлу.
- Рядок 6 (NEW): Дані з "Нового" файлу.
- Рядок 7+: Дані, зібрані з каталогу сканування.
